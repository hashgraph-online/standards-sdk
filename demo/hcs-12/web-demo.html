HCS-12 HashLinks Web Demo

Local demonstration of WASM action binding

⚠️ Prerequisites:
1. Build the WASM first: cd rust-wasm && ./build.sh
2. Serve this directory: npx http-server -p 8080
3. Open http://localhost:8080/web-demo.html

Note: This is a local demo showing how blocks bind to actions. For a full deployment to Hedera, run npm run demo:hcs12

3. Action Binding in Templates

In a real HashLink, the block template would look like:

<div class="counter-block"> <h2>Counter: {{attributes.count}}</h2> <div class="controls"> <button data-action="{{actions.0}}" data-params='{"amount": 1}'>+</button> <button data-action="{{actions.1}}" data-params='{"amount": 1}'>-</button> <button data-action="{{actions.2}}">Reset</button> </div> </div>

Where actions.0, actions.1, etc. reference the action aliases defined in the assembly.
